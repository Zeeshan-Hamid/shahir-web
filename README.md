# Ali Jan Website

A responsive static website built with HTML, CSS, and JavaScript.

## Features

- Fully responsive design that works on all devices
- Modern and clean UI
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Animated section reveals
- Contact form with validation
- Portfolio showcase with hover effects
- Skills progress bars

## Project Structure

```
.
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── script.js       # JavaScript functionality
└── README.md           # This file
```

## Usage

1. Clone or download this repository
2. Open `index.html` in your browser to view the website

## Customization

- Replace placeholder images with your own images
- Update text content in the HTML file
- Modify colors in the CSS file (look for `:root` variables)
- Add or remove sections as needed

## Browser Support

This website works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
